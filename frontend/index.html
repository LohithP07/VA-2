Why Veracity Agent

Everything you need for
document trust

A complete document integrity platform — not just a hash checker.

🔐

Cryptographic Signing

Every document is signed with RSA-PSS-SHA256 by a trusted issuer. Independent key pairs per issuer ensure zero cross-contamination.

⛓️

Local Blockchain Ledger

Each document hash is anchored as a block in a SHA-256 linked chain — providing immutable, tamper-evident provenance.

🏛️

Multi-Issuer Identity

DID-based issuer registry supports MLRITM, External CA, and Govt Registry — each with its own cryptographic identity.

🧠

AI Anomaly Detection

Isolation Forest ML model monitors upload and verification patterns in real-time, flagging unusual activity automatically.

🤖

Autonomous Response

When anomalies are detected, the autonomous agent automatically revokes credentials — no human intervention required.

👤

Per-User Isolation

Each account has its own private ledger. Your documents are yours alone — other users cannot see or access your records.

📋

Copy Detection

If a file's hash already exists in the system under another account, you'll be notified — helping track document provenance.

📦

Batch Operations

Register and verify multiple files in a single API call. Perfect for bulk document audits and large-scale deployments.

📊

Spreadsheet Parsing

Preview and validate Excel/CSV content on upload. Each sheet gets its own SHA-256 hash for granular integrity checking.

How it works

Four steps from upload to blockchain-anchored trust.

1

Create Account

Register with your email. Your account gets a private, isolated ledger that only you can access.

2

Upload Document

Upload any file — PDF, image, video, Excel, Word. Select the trusted issuer that will sign it.

3

Sign & Anchor

The system generates a SHA-256 hash, signs it with RSA-PSS, and anchors it to the blockchain ledger.

4

Verify Anytime

Upload the same file anytime — the system instantly tells you if it's Verified ✅, Tampered ⚠️, or Revoked 🔒.

About this project

Built for the age of digital trust

Veracity Agent was built to solve a real-world problem: how do you know a digital document hasn't been tampered with?

Using a combination of cryptographic signing, a local blockchain ledger, and machine learning anomaly detection, Veracity Agent provides end-to-end document integrity verification — from student transcripts to legal contracts.

Developed under the guidance of MLR Institute of Technology and Management (MLRITM), the system supports three DID-based trusted issuers and features an autonomous AI agent that can automatically revoke credentials when suspicious activity is detected.
