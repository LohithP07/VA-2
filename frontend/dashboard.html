🧠 AI Anomaly Detection

                        
Isolation Forest ML model monitors event patterns and triggers autonomous credential
                            revocation on suspicious activity.

                    


                    

                        
                        
                    


                    


                    

                        
How it works

                        

                            The anomaly detection module uses scikit-learn's Isolation Forest to
                            analyze upload and verification event patterns.
                            Features analyzed include request count and event type encoding.
                        

                        

                            If the latest event is flagged as an outlier (prediction == -1), the autonomous
                            agent can automatically revoke credentials for all unrevoked ledger records — protecting
                            against compromised or unusual activity.
